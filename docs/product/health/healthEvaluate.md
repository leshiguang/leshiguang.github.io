# 核心价值点
- **权威专家背书，六维度综合评价国人健康**

与国家工程院陈君石院士旗下的专业健康管理团队共研，参考美国密西根大学健康计分评估法，打造完全适合于中国人群生活方式的全面健康评估模型

- **企业便捷筛选目标用户的分析工具**

无需硬件，仅需3分钟填写问卷，即可获取用户的健康数据，锁定健康风险，找到目标用户

- **快速全面收集用户信息，进行精准化推荐决策**

结合用户健康风险标签，为用户提供个性化改善方案，并从专业健康角度提升商品转化<br />![幻灯片2.JPG](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622456113765-05f7f489-51c3-4775-bbcb-040b4e1964c3.jpeg#clientId=u77077569-8ecd-4&from=ui&id=O7NnB&margin=%5Bobject%20Object%5D&name=%E5%B9%BB%E7%81%AF%E7%89%872.JPG&originHeight=720&originWidth=1280&originalType=binary&size=163195&status=done&style=stroke&taskId=ud21e02da-2cca-4769-bded-8aae22275cd)

## 评估入口
![演示文稿11.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622190553390-43882355-f094-4dcb-b1d2-ab4fae78a22a.jpeg#clientId=u5b86eb70-a02b-4&from=drop&id=NH47U&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF11.jpg&originHeight=720&originWidth=1280&originalType=binary&size=51298&status=done&style=stroke&taskId=ud9cd4234-653b-4ae6-9fd7-9f31b3a6c3f)<br />全面风险评估入口，位于小程序首页上方，可展示引导文案或评估结果。

- 当用户未进行过全面风险评估时，该入口提示“3分钟评估健康指数”，点击后，可跳转至评估引导页，引导用户进行全面风险评估。
- 当用户已有全面风险评估历史时，该入口显示用户最近一次的健康生活指数及变化趋势，点击后，可跳转至评估结果页，用户可查看相关评估结果及解读。
- 当用户已有全面风险评估历史，且距离上次**手动更新**问卷日期超过28天时，该入口提示“太久未更新评估”，不显示最近一次的健康生活指数及变化趋势，点击后，可跳转至评估结果页，用户可查看相关评估结果并更新问卷。

## 评估结果页
![演示文稿11.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622191869538-054e70b6-912e-4f21-ae06-a2eaa733f273.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=Duw0P&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF11.jpg&originHeight=720&originWidth=1280&originalType=binary&size=178842&status=done&style=stroke&taskId=ufbc32c19-94a3-454b-ab32-b32fb4fcb2c)<br />评估结果页，可展示用户某次评估的健康生活指数、健康生活指数构成、健康年龄对比及解读、影响健康的风险因素（健康风险标签）、风险标签解读、改善方案及商品推荐、其他健康测评入口、分享功能等多项内容。
<a name="u7Ura"></a>
### 健康生活指数及构成
![演示文稿2.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622192153402-6dd9546b-626e-4e11-b98b-c756d0a23acf.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=LBT7o&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF2.jpg&originHeight=720&originWidth=1280&originalType=binary&size=37054&status=done&style=stroke&taskId=u859e5efa-b918-4254-b9ec-b302da52832)<br />该部分用于展示用户该次评估所产生的健康生活指数，及健康生活指数的构成。<br />健康生活指数，根据用户的答题状况或硬件产生的数据，由体重、睡眠/压力、运动、血压、血糖、膳食六部分组成。某一部分的得分越高，则代表用户在该部分的生活方式越健康，在图中点的位置也越靠外围。<br />​<br />
<a name="V2RyJ"></a>
### 健康年龄对比及解读
![演示文稿3.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622193599434-b1578b9b-8807-45a5-809c-f9a35b47b971.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=T8X1O&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF3.jpg&originHeight=720&originWidth=1280&originalType=binary&size=36334&status=done&style=stroke&taskId=u05f6fd0c-c22f-4463-a034-b8065a03e04)<br />该部分用于展示用户该次评估所产生的健康年龄、健康年龄与实际年龄的对比与解读。<br />健康年龄与用户的健康生活指数相关，健康年龄越小，则代表用户整体的生活方式越健康。同时，评估结果页提供健康年龄与实际年龄的对比与相关解读。<br />健康年龄解读分为两大类：单次测评解读与阶段性回顾解读

   - 阶段性回顾解读：每14天，结果页会根据用户目前的健康风险标签与14天前的健康风险标签进行对比，告知用户阶段性健康风险标签的变化情况。
   - 单次测评解读：其余时间，会根据用户目前的评估结果，对健康年龄进行详尽解读。



<a name="uss8A"></a>
### 影响健康的风险因素（健康风险标签）
![演示文稿4.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622194048699-5a594a45-e3ba-4741-96a7-2b1906fbaf52.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=gYjP9&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF4.jpg&originHeight=720&originWidth=1280&originalType=binary&size=34082&status=done&style=stroke&taskId=u9624c9c5-bad0-4645-9c0e-20ff8d09c01)<br />该部分用于展示用户所有的健康风险标签，标签来源包括但不限于：全面风险评估、其他评估、硬件测量数据、手动记录数据等。<br />该部分只展示用户“风险”或“疾病”类的健康标签，不展示“健康”类的健康标签。<br />​<br />
<a name="KxMFT"></a>
### 风险标签解读
![演示文稿5.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622194349543-ab311335-406f-4f45-8b0b-071f536c29a6.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=L1fbn&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF5.jpg&originHeight=720&originWidth=1280&originalType=binary&size=83804&status=done&style=stroke&taskId=ueb79fc48-8339-4292-8ca8-3b81dc28cec)<br />该部分根据用户命中的健康风险标签，给予对应的解读。解读时，根据各标签所属的业务分类，按照“体重、睡眠/压力、运动、血压、血糖、膳食”的顺序进行解读，若某某块无对应健康风险，则不展示该模块的解读。<br />解读包括生活方式建议、保健品建议、行动建议等。<br />​<br />
<a name="r4sg8"></a>
### 改善方案及商品推荐
![演示文稿6.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622202584044-86f73f75-f863-4cf7-a1d7-05a6f913b114.jpeg#clientId=u0937555a-9e20-4&from=ui&id=GYCwm&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF6.jpg&originHeight=720&originWidth=1280&originalType=binary&size=57971&status=done&style=stroke&taskId=u1ac5a650-a93a-4c3f-a188-fe806eb6a09)<br />该部分根据用户命中的健康风险标签，根据各业务指定的推荐逻辑，可为用户推荐对应业务的改善方案及商品推荐。用户点击改善方案或商品，可跳转到对应的方案二级页或商品详情页。<br />​<br />

### 其他健康测评入口
![演示文稿7.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622195054961-f640a673-0667-4718-a422-ff37841b59a2.jpeg#clientId=u5b86eb70-a02b-4&from=ui&id=ANimr&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF7.jpg&originHeight=720&originWidth=1280&originalType=binary&size=30168&status=done&style=stroke&taskId=u674719cb-b52a-42c1-a26f-0257f312c07)<br />在评估结果页中，可展示其他健康测评的入口，用户通过点击，可跳转到相关的健康测评页面。<br />目前暂时只接入免疫力评估一项。<br />​<br />
<a name="YTyzV"></a>
### 分享功能
![演示文稿8.jpg](https://cdn.nlark.com/yuque/0/2021/jpeg/2752327/1622202863432-defd4431-4034-4687-9473-52a60d219ed0.jpeg#clientId=u0937555a-9e20-4&from=ui&id=m9El3&margin=%5Bobject%20Object%5D&name=%E6%BC%94%E7%A4%BA%E6%96%87%E7%A8%BF8.jpg&originHeight=720&originWidth=1280&originalType=binary&size=137889&status=done&style=stroke&taskId=u8784a71f-2647-437a-9102-a50e342dd4e)<br />在评估结果页中，提供两种分享能力：“炫耀一下”与“分享给亲友”

   - 炫耀一下：当用户该次评估计算得出的健康年龄小于实际年龄时，会出现“炫耀一下”按钮。点击后，会生成带有健康生活指数、健康年龄对比及解读、小程序码的图片。用户可长按图片保存并分享。若用户挨次评估计算得出的健康年龄大于等于实际年龄时，不出现“炫耀一下”按钮。被分享的用户，可以通过长按图片识别小程序码的方式，跳转至乐智健康小程序。
   - 分享给亲友：在评估结果页最下方，无论用户的健康年龄、健康生活指数如何，均出现“分享给亲友”按钮。点击后，会引导用户以小程序卡片的方式，将乐智健康小程序分享给亲友。被分享的用户，可以通过点击小程序卡片的方式，跳转至乐智健康小程序。

# 产品交付能力
<a name="xdSog"></a>
## 硬件接入角度
无论用户是否绑定/使用硬件，自用户完成首次评估的第二天起，每日10点健康全面风险评估均会根据前一天的健康数据（含问卷来源数据、手动记录数据、硬件上传数据），自动更新全面风险评估结果。
<a name="G9czo"></a>
### 有硬件交付版本
若用户有绑定一个或多个硬件（包括但不限于手环、秤、血压计、血糖仪），用户在当天由硬件产生的健康数据，会影响该用户第二天的健康生活指数及健康年龄。
<a name="GkdMR"></a>
### 无硬件交付版本
若用户未绑定任何硬件，仍然可以正常使用全面风险评估的所有功能。<br />用户通过手动记录的健康数据，同样会影响该该用户第二天的健康生活指数及健康年龄。<br />​<br />
<a name="fGB6E"></a>
## 业务接入角度
<a name="qDT3x"></a>
### 接入全部业务
若客户接入了乐智健康小程序的全部业务，则可以正常使用全面风险评估的全部功能。
<a name="Z565A"></a>
### 接入部分业务
若客户只接入了乐智健康小程序的部分业务，则会影响“改善方案推荐”的展示。该部分的推荐展示，需要根据客户的实际接入情况，决定显示或者隐藏。

   - 若客户未接入血压业务，则改善方案推荐中，血压模块的“控压计划”推荐位需要隐藏；
   - 若用户未接入血糖业务，则改善方案推荐中，血糖模块的“控糖计划”推荐位需要隐藏；
   - 若客户未接入体重业务，则改善方案推荐中，体重模块的“体重管理方案”推荐位需要隐藏；
   - 若客户未接入运动业务，则改善方案推荐中，运动模块无需修改（目前运动模块无推荐）；
   - 若客户未接入营养管理业务，则改善方案推荐中，膳食模块的“膳食均衡评估”推荐位需要隐藏；
   - 若客户未接入睡眠业务，则改善方案推荐中，血压模块的“睡眠改善方案”/“减压放松改善方案”推荐位需要隐藏；
   - 若客户未接入免疫力业务，则结果页最下方的“健康测评”模块需要整体隐藏。

## 商品接入角度

# Q&A
Q：全面风险评估测评的结果可靠吗？<br />A：全面风险评估所使用的模型是由乐智健康与国家工程院陈君石院士旗下的专业健康管理团队共同打造的，拥有权威机构背书认证，更适合国人，结果准确可靠。<br />​

Q：全面风险评估测评需要自己手动更新吗？<br />A：如果您拥有相关的智能设备，如手环、体重秤等，小程序会每天定时同步相关设备数据并动态更新您的风险测评结果，无需额外的手动更新。但为了让您的测评结果更加适合您的当前动态，建议您定时对饮食、饮酒等数据进行更新。<br />​

Q：全面风险评估测评一定要有手环/体重秤等设备才能做吗？<br />A：使用相关的测量设备，可以让您评估结果得到实时更新并且更加准确可靠。但是若您暂时无法购买使用相关设备，您仍可以通过手动输入相关数据的方式完成测评。<br />​

Q：评估结果对我有什么用？<br />A：全面风险评估可以让您对当前的生活方式有一个全面的了解，您可以根据测评结果所提供的建议，对您生活方式中的不足进行改正，减少相关风险，获得更健康的身体素质。<br />​
