# 概述
<a name="zlubH"></a>
## 关于SDK
Android SDK实现乐心设备私有蓝牙协议的接入工作，支持手环、体脂秤和血压计等设备，提供绑定、配对、设置、数据解析和数据上传等功能， SDK中实现了蓝牙通信中容易出现的丢包粘包的问题，解决了蓝牙连接池的管理、连接速度慢和连接不稳定等问题，整体采用领域驱动设计的设计思想实现。
